Featured Project

Project Community

Goal / Mission

Project Community is a social services / community resources application with the goal of transforming lives by helping people find resources nearby. As a caseworker with the state of WA, our mission was to transform lives. The mission of Project Community is an attempt to continue transforming lives by helping people in a more efficient and larger scale than being an individual caseworker. Although I may not be able to help each and every person personally, Project Community can point people in the right direction and can be a large impact on a person's life.

Design / Layout

Project Community was developed with access and ease of use above all else. With unambiguous straight to the point interactions / feedback users are able to access the info they are looking for in a clean and expedited manner.

Technology Used

Project Community is built using HTML, CSS, JavaScript, Python, and Django for the views, routes, models and PostgreSQL to manage the database. Full CRUD functionality was built along with an emphasis on developing for location data/Google Maps.

Language Accessibility

To ensure language is not a barrier preventing those in need Google Translate is utilized and easily accessible from any page.

Vision Impairment

Custom code was built to allow easy adjustment of font size and line height from any page as well. This works together with Google Translate to ensure many demographics are able to use Project Community.

Responsive Technology

All pages were designed to be responsive. Whether on a phone, tablet, laptop, or desktop Project Community should be functional from any device a user has access to ensuring users are not locked out of use due to device limitations.
